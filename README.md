# TMDB Movies UI

A movie browsing UI inspired by TMDB.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- TMDB API

## Features
- Browse popular & trending movies
- Search movies by title
- Wishlist using localStorage
- Responsive UI
- Clean component structure

## Setup
1. Clone repo
2. Run `npm install`
3. Create `.env.local`
4. Add TMDB API key
5. Run `npm run dev`
